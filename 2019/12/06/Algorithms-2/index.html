最长公共子序列

题目

找出两个字符串中最长公共子序列（非最长连续子序列）

解决方案

  • 动态规划：写出状态方程

    • 当i=0，j=0时，c[i,j] = 0
    • 当a[i] = b[j]时，c[i,j] = c[i-1, j-1] + 1
    • 当a[i] 不等于 b[j]时，c[i,j] = max(c[i,j-1], c[i-1, j])

KMP

题目

判断字符串A是否是字符串B的子串

解决方案

  • 指针a指向字符串A，指针b指向字符串b，next数组为部分匹配表（字符串A的前缀后缀最长公共序列长度）
  • 若a尚未匹配到元素
    • a指向的内容若不等于b指向的内容，则b++
    • 若相等，二者同时++
  • 若a已有匹配元素
    • 若a指向的内容若不等于b指向的内容
      • b += （a - next[a-1])
      • a = 0
    • 若相等，二者同时++
